# 试题说明

客户 A 公司想要申请一件实用新型专利, 发现竞争对手 B 公司拥有同类专利, 现委托你所在的代理机构对 B 公司的实用新型专利附件 1(下称涉案专利) 提出无效宣告请求。A 公司向你所在的代理机构提供了涉案专利及其美国优先权文件译文(附件 2), 以及 A 公司所研发的技术的交底材料。你自行检索了三份对比文件, 现委托你所在的专利代理机构提出无效宣告请求, 并规避涉案专利可能的侵权风险。

第1题:请你对客户提供的材料以及检索到的文件进行分析,并撰写一份无效宣告请求书,在无效宣告请求书中要明确无效宣告请求的范围、理由和证据,要求以《专利法》及其实施细则中的有关条、款、项作为独立的无效宣告理由提出,并结合给出的材料具体说明。

第2题:请你根据A公司所研发产品的技术交底材料,综合考虑客户提供的涉案专利和对比文件1-3所反映的现有技术,为客户撰写一份实用新型专利申请的权利要求书。如果分案,请说明分案的理由。

第3题:具体描述你撰写的独立权利要求相对于现有技术具备新颖性和创造性的理由。如有多项独立权利要求,请分别对比和说明。

第 4 题: 根据涉案专利和对比文件 1-3, 撰写一份实用新型专利完整的说明书, 针对技术交底材料的补充和修改简要说明理由。

# 自吸水花盆

[0001]本实用新型涉及一种自吸水花盆。

[0002]现有技术中的双层花盆可以在底层储水,但是水位高于内盆体的底部时,容易致使植物根部腐烂;水位低于内盆体的底部时,容易致使植物干旱,不利于植物生长。

[0003]本实用新型要解决的技术问题是双层花盆上层土壤无法自动从底层储水腔吸水的问题。

[0004]本实用新型提供一种花盆，包含内盆体、外盆体和托盘，内盆体的底部上设有若干个细孔和至少一个通孔，内盆体与外盆体之间形成储水腔，还包括一个吸水件，其穿过通孔置于内盆体内，吸水件的至少一个端部接触外盆体的底部，外盆体的侧壁上设有溢水口，溢水口的位置不高于内盆体的底部。

[0005]本实用新型通过设置一个吸水件,连通底层储存的水和上层土壤,达到花盆自吸水为植物浇水的效果。

[0017]浇水时可以直接浇入内盆体1内，多余的水可透过内盆体1的底板10，储存于储水腔5内；也可以通过设置在外盆体2侧壁上的注水口22注水，水直接进入储水腔5内储存，通过吸水件4吸入内盆体1为植物所吸收。

[0018]为了便于知道何时应该浇水，可以在外盆体2的侧壁上设置水位观测装置。水位观测装置可以是如图5所示的开设在外盆体2侧壁上的透明条状观察窗6，观察窗6上设置水位刻度，方便观察储水腔5内的水量；水位观测装置也可以是放入外盆体2和内盆体1之间的水位计(图中未示出)。

# 一种透气双层花盆

[0001]现有技术中的普通花盆透气性差,不利于植物生长,为了提高透气性,本实用新型提供了一种透气双层花盆,提高了透气性,有利于植物健康生长。

[0002]如图1-2所示,本实用新型的透气双层花盆包括外盆1,内盆2,吸水管3,内盆2的上边缘搭置在外盆1的上边缘处,内盆2的底部与外盆1的底部之间具有容水空腔4。内盆2的底部具有穿孔,吸水管3穿过内盆2底部的穿孔,其一端伸入内盆2中,另一端伸入容水空腔4内并与外盆1的底部接触。外盆1的侧壁上开设有至少一个透气孔13,内盆2的侧壁上开设有至少一个透气孔21。

[0003]外盆1的侧壁上开设有漏水孔11,漏水孔11位于内盆2的下方。进一步的,为了接住漏水孔11的水,可以设置一个托盘(图中未示出),漏出水流在托盘上,以防止漏水流到外面。为方便浇灌,外盆1的侧壁上还开设有注水口12。

# 技术交底材料：

现有技术中的双层花盆透气性差，不利于植物的健康生长；且随着双层花盆底部储水腔中的水分渐干，容易导致植物干旱，若将储水腔中的水加满至上层土壤中，又容易造成植物根部泡水而腐烂。为了解决上述问题，本方案提供了一种花盆，既利于透气又利于吸水。

如图1、图2所示，该花盆包括内盆体1、外盆体2和托盘（图中未示出），内盆体1的底部与外盆体2的底部之间设有储水腔3。内盆体1的侧壁上开设有至少一个内透气孔（图中未示出），外盆体2的侧壁上开设有至少一个外透气孔（图中未示出）。内盆体1的底部具有底板11，底板11上开设有至少一个通孔12和多个渗水孔13，还包括一个吸水件4，吸水件4穿过通孔12，其一端伸入内盆体1，另一端伸入储水腔3并接触外盆体2的底部。

进一步地，外盆体2的侧壁上低于内盆体1底部的位置开设有溢水孔21，储水腔3内的水可由溢水孔21流出以防止储水腔3内的水过满导致植物根部腐烂，托盘可用于当花盆放置在室内时接收由溢水孔21流出的水，当花盆放置在室外时，也可以不设置托盘。外盆体2的侧壁上高于内盆体1底部的位置还开设有注水孔22，可通过注水孔22直接向储水腔3内蓄水用于浇灌。

由于花盆内、外盆体的侧壁上分别开设有内、外透气孔，在提高花盆透气性的同时，会导致土壤失水。为解决水分散发快的问题，该花盆提供了第二实施例，如图3所示，其与第一实施例的区别在于底板11上开设有至少两个通孔12，吸水件4的两端分别穿过通孔12伸入储水腔3并接触外盆体2的底部，在内盆体1的底板11上设置向上凸起的凸台5，吸水件4位于内盆体1内的部分搭设在凸台5上，凸台5具有支撑作用，增大了吸水件4与土壤的接触面积，使土壤内的供水充分且均匀。

如图4所示，第三实施例与第二实施例的区别在于在内盆体1的底板11上设置架体6，架体6可以是桥状架体或板状架体。以桥状架体为例，包括桥体61和透水部62，透水部62沿着桥体61间隔设置，透水部62可以是孔状或槽状，吸水件4位于内盆体1内的部分穿过桥体61，架体6具有支撑作用，吸水件4通过透水部62与土壤接触。

如图5所示，第四实施例与第二实施例的区别是在内盆体1的底板11上设置至少两根套管7，套管7分别固定插入在通孔12内，套管7上开设有均匀分布的细孔71，至少两个吸水件4位于内盆体1内的部分分别穿过套管7，套管7具有支撑作用，吸水件4通过细孔71与土壤接触。

进一步地，还设置有水位观测装置（图中未示出），水位观测装置可以是设置在外盆体2侧壁上的透明条状观察窗，方便观察储水腔3内的水位，也可以是放置在外盆体2和内盆体1之间的水位计。

进一步地,还设置有湿度计(图中未示出),用于检测土壤的实时湿度,当湿度不符合设定条件时,通过指示灯进行报警,提示及时浇水。

# 第一题

尊敬的国家知识产权局：

根据《专利法》第45条及《专利法实施细则》第69条的规定，本请求人现请求宣告专利号为ZL202025634028.X、名称为“自吸水花盆”的实用新型专利(以下简称涉案专利)无效，具体理由如下：

# 一、关于证据的使用

请求人提交的证据为：

附件1:无效宣告请求针对的实用新型专利ZL202025634028.X,授权公告日为2020年12月11日,申请日为2020年3月18日,优先权日为2019年10月8日;

附件2:ZL202025634028.X实用新型专利的优先权文件译文；

对比文件 1: 实用新型专利 ZL201620345678.9, 授权公告日为 2016 年 8 月 6 日, 申请日为 2016 年 6 月 25 日;

对比文件2:实用新型专利ZL201620789117.7,授权公告日为2017年3月23日,申请日为2016年9月4日;

对比文件3:实用新型专利ZL201920789125.6,授权公告日为2020年1月10日,申请日为2019年8月10日;

对比涉案专利和附件2可知，涉案专利权利要求1-4的技术方案已经记载在附件2的说明书中，两者技术领域、所解决的技术问题、技术方案和预期效果均相同，属于相同主题的发明创造，且涉案专利的申请日(2020年3月18日)距其所要求的优先权日(2019年10月8日)在12个月之内，因此涉案专利的权利要求1-4可以享有附件2的优先权。附件2中没有记载涉案专利权利要求5所要求保护的“外盆体的侧壁上设置有注水口”的技术内容，因此涉案专利权利要求5不能享有附件2的优先权。

对比文件1、对比文件2的授权公告日早于涉案专利的优先权日，均属于现有技术；对比文件3的申请日早于涉案专利的优先权日，授权公告日晚于涉案专利的优先权日，属于申请在先、公开在后的中国专利文件，仅可用于评价涉案专利权利要求1-4的新颖性。而对于权利要求5而言，对比文件3属于现有技术。

# 二、无效宣告请求理由

# 1. 权利要求 1 相对于对比文件 3 不具备新颖性

权利要求1请求保护一种花盆，对比文件3第[0002]段、[0003]段以及图1、2中公开了一种透气双层花盆，包括外盆1，内盆2，吸水管3，内盆2的底部与外盆1的底部之间具有容水空腔4（相当于权利要求1中的内盆体与外盆体之间形成储水腔）。内盆2的底部具有穿孔（相当于权利要求1中的内盆体的底部上设有通孔），由图中可看出内盆的底部具有若干细孔，吸水管3穿过内盆2底部的穿孔，其一端伸入内盆2中，另一端伸入容水空腔4内并与外盆2的底部接触。外盆1的侧壁上开设有漏水孔11（相当于权利要求1中的溢水口），漏水孔11位于内盆2的下方。为接住从漏水孔11流出的水，可以设置托盘。

由此可见，对比文件3公开了权利要求1的全部技术特征，二者相比，技术方案实质相同，两者属于相同的花盆技术领域，解决相同的技术问题并具有相同的预期效果，均是提供一种当水量过少时可向土壤内吸水、防止植物干旱，当水量过多时可向外溢水、防止植物腐烂的花盆。因此对比文件3构成涉案专利权利要求1的抵触申请，权利要求1相对于对比文件3不具备新颖性，不符合《专利法》第22条第2款的规定。

# 2. 权利要求1相对于对比文件1和对比文件2结合不具备创造性

对比文件2的技术领域与涉案专利相同,公开的技术特征最多,所要解决的技术问题最为接近,因此对比文件2是最接近的现有技术。

对比文件2在说明书第[0002]段、图1中公开了一种自吸水花盆，包括内盆体1，外盆体2和吸水件3，内盆体1的底部外侧和外盆体2的底部内侧之间设有储水腔4。内盆体1的底部为板状，内盆体1的底板上设有若干个细孔和至少一个通孔，吸水件3穿过通孔，其一端伸入内盆体1内，另一端可以接触外盆体2的底部；外盆体的侧壁上开设有溢水口21，从图中可以看出，溢水口低于内盆体底部。

权利要求1所要求保护的技术方案与对比文件2公开的技术内容相比，区别在于：花盆还包括托盘。

基于上述区别技术特征可知,权利要求1实际所要解决的技术问题是提供一种可接收溢出的水,避免污染桌面的花盆。

对比文件1在说明书[0004]段公开了一种花盆，可以将盆体1放置在托盘上，用于接住溢水孔12漏出的水。由此可见，上述区别技术特征已被对比文件1公开，且该特征在对比文件1中所起的作用与其在权利要求1中所起的作用相同，均是用于接住溢水孔漏出的水，避免污染桌面。

对比文件1给出了将上述区别技术特征与对比文件2结合以解决上述技术问题的启示,权利要求1相对于对比文件2和对比文件1的结合是显而易见的,不具有实质性特点和进步,不具备创造性,不符合《专利法》第22条第3款的规定。

# 3. 权利要求2相对于对比文件1和对比文件2结合不具备创造性

从属权利要求2的附加技术特征进一步限定了“吸水件(4)为呈倒U形的吸水附件(41)”。对比文件2在说明书[0002]段公开了“吸水件3也可以设置为弯曲的倒U形”，上述技术特征在对比文件2中所起的作用与其在权利要求2中为解决其技术问题所起的作用相同，都是防止吸水件从内盆体里面滑脱出来，同时增加吸水件与土壤的接触面积。因此在其所引用的权利要求1不具备创造性的情况下，权利要求2相对于对比文件2和对比文件1的结合也不具备创造性，不符合《专利法》第22条第3款的规定。

# 4. 权利要求3引用权利要求1时的技术方案不清楚

从属权利要求3引用权利要求1或2，但权利要求3中的“倒U形吸水附件”在权利要求1中并无记载。因此从属权利要求3引用权利要求1时缺乏引用基础，导致权利要求3的保护范围不清楚，不符合《专利

法》第26条第4款的规定。

# 5. 权利要求4相对于对比文件1和对比文件2结合不具备创造性

从属权利要求4的附加技术特征进一步限定了“还包括水位观测装置”。对比文件2在说明书[0002]段公开了“外盆体2的内侧壁和内盆体1的外侧壁之间设置有水位计5”，其中水位计为水位观测装置的下位概念。上述技术特征在对比文件2中所起的作用与其在权利要求4中所起的作用相同，都是用于方便观察储水腔内的水量。因此在其所引用的权利要求2不具备创造性的情况下，权利要求4相对于对比文件2和对比文件1的结合也不具备创造性，不符合《专利法》第22条第3款的规定。

# 6. 权利要求5相对于对比文件3不具备新颖性

权利要求 5 不能享有优先权, 因而判断新颖性时以申请日为准, 而对比文件 3 的公开日早于权利要求 5 的申请日, 对比文件 3 是现有技术, 可以评价权利要求 5 的新颖性。

权利要求 5 是权利要求 1 的从属权利要求, 其附加技术特征是“外盆体(2)的侧壁上设置有注水口(22)”。

对比文件3在说明书[0003]段以及图1公开了“外盆1的侧壁上还开设有注水口12”，由此可见该附加特征被对比文件3公开，因此当其引用的权利要求1不具备新颖性时，该从属权利要求5也不具备《专利法》第22条第2款所规定的新颖性。

综上所述，现请求宣告专利号为“ZL202025634028.X”、名称为“自吸水花盆”的实用新型专利的权利要求1、2、4、5以及权利要求3引用权利要求1的技术方案无效。（或写为部分无效）

# 花盆

# 技术领域

本实用新型涉及一种花盆，尤其涉及一种在透气的同时防止水分散失过快的花盆。

# 背景技术

现有技术中的双层花盆透气性差，不利于植物的健康生长；且随着双层花盆底部储水腔中的水分渐干，容易导致植物干旱，若将储水腔中的水加满至上层土壤中，又容易造成植物根部泡水而腐烂。

申请号为201920789125.6的实用新型专利公开了一种透气双层花盆，包括外盆，内盆，吸水管，内盆与外盆的底部之间具有容水空腔。内盆的底部具有穿孔，吸水管穿过内盆底部的穿孔，其一端伸入内盆中，另一端伸入容水空腔内并与外盆的底部接触。外盆和内盆的侧壁上均开设有透气孔。外盆的侧壁上位于内盆下方开设有漏水孔。

上述透气双层花盆不仅可以防旱防涝，且提高了透气性，但由于仅设置了透气孔，花盆透气的同时水分散失过快，容易导致植物缺水干旱。

# 实用新型内容

本实用新型所要解决的问题是提供一种既利于透气又利于吸水的花盆。

为解决上述技术问题,本实用新型提供了一种花盆,包括内盆体和外盆体,内盆体的底部与外盆体的底部之间设有储水腔,内盆体的底部具有底板,底板上开设有渗水孔和至少一个通孔,内盆体的侧壁上开设有至少一个内透气孔,外盆体的侧壁上开设有至少一个外透气孔,还包括吸水件,吸水件穿过通孔,其至少一端伸入储水腔并接触外盆体的底部、另一端伸入内盆体,底板上设置有支撑部件,用于支撑吸水件位于内盆体内的部分。

体内的部分，使得花盆透气的同时又能使土壤内的供水充分且均匀，避免植物缺水干旱。

# 具体实施方式

如图1、图2所示，该花盆包括内盆体1、外盆体2和托盘（图中未示出），内盆体1的底部与外盆体2的底部之间设有储水腔3。内盆体1的侧壁上开设有至少一个内透气孔（图中未示出），外盆体2的侧壁上开设有至少一个外透气孔（图中未示出）。内盆体1的底部具有底板11，底板11上开设有至少一个通孔12和多个渗水孔13，还包括一个吸水件4，吸水件4穿过通孔12，其至少一端伸入储水腔3并接触外盆体2的底部、另一端伸入内盆体1。

进一步地，外盆体2的侧壁上低于内盆体1底部的位置开设有溢水孔21，储水腔3内的水可由溢水孔21流出以防止储水腔3内的水过满导致植物根部腐烂，托盘可用于当花盆放置在室内时接收由溢水孔21流出的水，当花盆放置在室外时，也可以不设置托盘。外盆体2的侧壁上高于内盆体1底部的位置还开设有注水孔22，可通过注水孔22直接向储水腔3内蓄水用于浇灌。

由于花盆内、外盆体的侧壁上分别开设有内、外透气孔，在提高花盆透气性的同时，会导致土壤失水。为解决水分散发快的问题，该花盆提供了第二实施例，如图3所示，其与第一实施例的区别在于底板11上开设有至少两个通孔12，吸水件4的两端分别穿过通孔12伸入储水腔3并接触外盆体2的底部，在内盆体1的底板11上设置向上凸起的凸台5，吸水件4位于内盆体1内的部分搭设在凸台5上，凸台5具有支撑作用，增大了吸水件4与土壤的接触面积，使土壤内的供水充分且均匀。
