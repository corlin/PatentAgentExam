二、专利收费标准调整后，继续对确有困难的职务发明（单位）和非职务发明（个人）的收费实行部分减缓。减缓项目仍为申请费、发明维持费、发明审查费、复审费和授权以后三年的年费等五项。实行减缓的具体条件按国家知识产权局制定的有关规定。
